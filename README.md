## 🚀 Features

- 🌍 250+ Countries
- 🏛️ States & Provinces
- 🏙️ Cities Database
- 🌐 Multi-language support (18+ languages)
- ⚡ Ultra-fast JSON access
- 📦 Fully offline (no API required)
- 🔌 Works with Node.js / React / Flutter backend
